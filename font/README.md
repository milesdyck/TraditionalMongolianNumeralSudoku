# Khatan_Numerals

Khatan Numerals is an OpenType font containing custom glyphs for the Mongolian digit code points U+1811 to U+1819. Copyright (c) 2026 Altanzaya Yansanjav,
with Reserved Font Name "Khatan Numerals".
